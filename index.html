Core SIP Formulas

1. Monthly Rate: r = (1 + R_annual)^(1/12) − 1
2. Step-Up SIP: PMT_y = PMT_1 × (1 + g)^(y−1) [g = annual step-up]
3. Monthly engine: Portfolio_t = (Portfolio_t-1 + PMT) × (1 + r)
4. Total Invested: Σ [ PMT_1 × (1+g)^(y−1) × 12 ] for y = 1 … N
5. Total Gains: Portfolio_final − Total Invested
6. Real Value: FV_real = FV_nominal / (1 + π)^N [π = inflation]
7. Wealth Mult: WM = FV_nominal / Total Invested
8. Effective CAGR: (FV / Invested)^(1/N) − 1

Goal Planner

Binary search on PMT_1 so that Step-Up SIP FV equals Target
Convergence: Rs. 1 precision · max 200 iterations
Range: lo=Rs.1, hi=Target (self-bounding upper limit)

FIRE Number

Future monthly expenses = Monthly × (1 + π)^N
Future annual expenses = Future monthly × 12
FIRE Number (nominal) = Annual expenses / Withdrawal rate
FIRE Number (real) = Nominal / (1 + π)^N
Sustainable income/yr = FIRE Number × Post-retirement return

Key Assumptions

• SIP invested at END of each month (ordinary annuity)
• Annual return converted to geometric monthly equivalent
• Step-up applied at start of each calendar year
• Inflation compounds annually on the final nominal corpus
• Pre-tax returns shown — no CGT or withholding tax deducted
• Suitable for PSX KMI-30 / Equity MF / Shariah-compliant context
